# HexPuzzle
VR-игра головоломка "HexPuzzle", в которой игроку необходимо верно устанавливать кубы с частями изображений, собирая цельную картинку.
Для запуска уровня выберите пункт "Начать игру", выберите основные уровни или пользовательский уровень. В основных уровнях выберите один из доступных уровней.
В пользовательском уровне, выберите нужную локацию, включите/выключите таймер, выберите изображение, которое необходимо собрать, установите уровень сложности -> нажмите играть.
Чтобы поставить игру на паузу, нажмите на кнопку, отвечающую за действие PauseButton в SteamVR
